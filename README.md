# 🌦 Weather Analysis Project 

## 📌 Project Overview

This project focuses on analyzing weather data using Power BI.
It helps in understanding temperature patterns, rainfall trends, and seasonal changes through an interactive dashboard.

## 🎯 Objective

* To analyze historical weather data
* To identify trends and patterns
* To create an interactive and user-friendly dashboard

## 📊 Tools Used

* Microsoft Power BI

## 📁 Dataset

The dataset includes weather-related information such as:

* Temperature
* Humidity
* Rainfall
* Wind Speed

## 📈 Key Analysis Performed

* Temperature trend analysis
* Rainfall distribution
* Seasonal comparison
* Data visualization using charts and graphs

## 📊 Dashboard Features

* Interactive visuals
* Filters for better insights
* Easy-to-understand charts
* Clean and structured layout

## 🔍 Key Insights

* Identified temperature variations across seasons
* Observed rainfall patterns over time
* Found trends in weather conditions



---

## ⭐ Conclusion

This project demonstrates how Power BI can be used to transform raw weather data into meaningful insights for better decision-making.
